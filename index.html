iOctocat

… is a GitHub client for the iPhone and iPod Touch.
It will be available on the App Store the next days…

Features

Currently the application offers the following features:

  • My Private News Feed
  • My Activity Feed
  • Profile View
  • My Repositories
  • Repository Details
  • ... and counting
